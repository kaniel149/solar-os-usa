Type System

Typography

Two fonts. Clear hierarchy. Maximum readability.

DM Sans — Headlines

Marketing headlines, hero text, display. Warm and distinctive personality.

Display Solar OS Bold / 48px
H1 Page Title Bold / 36px
H2 Section Head Bold / 30px

Inter — UI & Body

Interface text, body copy, labels. Clean and highly legible at all sizes.

H3 Sub-section Semibold / 24px
Body Regular body text for paragraphs Regular / 16px
Caption METADATA & LABELS Medium / 12px
Overline SECTION LABEL Semibold / 12px

Typography Rules

  • Never use more than 2 font families in a single view
  • No text smaller than 12px — ever
  • No thin font weights (300 or below) for body text
  • Monospace (JetBrains Mono) for code, API docs, and system IDs only
