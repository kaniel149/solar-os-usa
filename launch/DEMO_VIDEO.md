## Screen Recording Instructions

### Setup
1. Use Mac screen recording (Cmd+Shift+5) or OBS for crisp 1080p
2. Set browser to full screen, hide bookmarks bar
3. Use Chrome with a clean profile (no extensions visible)
4. Set display to 1920×1080 resolution
5. Prep all pages in advance — no loading screens during recording

### Recording Order
Record each section separately (easier to edit):
1. **Messy desktop shot** — Open 5 tabs, make it look chaotic (10 sec)
2. **Dashboard** — Slow mouse movement across key metrics (10 sec)
3. **Lead Management** — Drag one card across the kanban board (10 sec)
4. **Permit Generator** — Select county, click generate, show PDF (10 sec)
5. **Proposal Builder** — Scroll through, hover on send button (10 sec)
6. **Monitoring** — Show production graph, zoom out to multi-site (10 sec)

### Tips
- Move the mouse slowly and deliberately — no jerky movements
- Pause 1 second on each key element before moving
- Use dummy data that looks real (real FL county names, realistic kW values)
- Keep the cursor visible but not distracting

## Voiceover Options

**Option A: Record yourself (recommended for authenticity)**
- Use Voice Memos on iPhone in a quiet room
- Speak slightly faster than normal conversation — 60 seconds is tight
- Practice 3x before recording
- Edit in iMovie or CapCut

**Option B: AI voiceover (faster, more polished)**
- Use ElevenLabs with a warm, confident male voice
- Model: "Josh" or "Adam" — professional but not corporate
- Generate per-section for easier timing
