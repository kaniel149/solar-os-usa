# Google Cloud for Startups

## Program URL
https://cloud.google.com/startup/apply

## What You Get
### Start Tier (our path — no VC required)
- Up to $2,000 in Google Cloud credits (1 year validity)
- Access to Google Cloud services
- Mentorship and startup resources

### Scale Tier (requires institutional funding)
- $100,000–$350,000 in credits
- AI-focused track available (Vertex AI, Gemini)
- Enhanced support and mentorship

## Eligibility
### Start Tier
- Technology startup
- NOT funded by institutional investors (we qualify!)
- Founded within last 5 years
- No Google Cloud credits beyond free trial

### Scale Tier (future)
- Equity funding from institutional investors
- Series A within last 12 months
- Founded within last 10 years
- ≤$5,000 prior Google Cloud credits

## Application Steps
1. Create a Google Cloud Platform (GCP) account
2. Set up a Billing ID
3. Go to https://cloud.google.com/startup/apply
4. Select **Start / Pre-funded** tier
5. Fill out:
   - Company info (KANIEL TORD LLC)
   - Founding date
   - Funding status: Self-funded / Pre-revenue
   - Team size
   - Website: kanielt.com
   - Cloud usage plans
6. Submit and wait 1-2 weeks for review

## What to Write in the Application

**Company Description:**
KANIEL TORD LLC is building Solar OS, an all-in-one cloud platform for the solar energy industry. We're consolidating CRM, project management, proposal generation, and installation tracking into a single SaaS solution for solar companies across the United States.

**Cloud Usage Plans:**
We plan to leverage Google Cloud for:
- Cloud Run / GKE for backend microservices
- BigQuery for analytics and reporting
- Vertex AI for intelligent proposal generation and energy production forecasting
- Cloud Storage for document and media management
- Firebase for real-time features and authentication

**Why Google Cloud:**
Google Cloud's AI/ML capabilities (especially Vertex AI and Gemini) are ideal for our intelligent features — automated proposal generation, satellite imagery analysis for roof assessments, and energy production predictions.

**Growth Plans:**
Starting with the US solar market, we aim to serve 100+ solar companies in year one. The platform will process thousands of proposals, manage installation projects, and provide data-driven insights to improve solar company operations.
