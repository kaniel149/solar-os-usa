# Microsoft for Startups Founders Hub

## Program URL
https://www.microsoft.com/startups

## What You Get
### Founders Hub (self-serve, no VC required)
- $25,000 in Azure credits (1 year validity)
- GitHub Enterprise (20 seats)
- Visual Studio Enterprise (5 seats)
- Microsoft 365
- Dynamics 365
- Power Apps
- OpenAI Service access

### Microsoft for Startups (higher tier, needs referral)
- Up to $150,000 in Azure credits (2 years)
- Requires partner/investor referral

## Eligibility
- Active startup under 10 years old
- Privately held
- Valid company email domain (kanielt.com ✓)
- Planning to build on Azure
- No VC funding required
- No accelerator affiliation required
- No existing Azure usage needed

### NOT Eligible
- Government entities
- Education institutions
- Consultancies
- Non-software products

## Application Steps
1. Go to https://www.microsoft.com/startups or partner.microsoft.com
2. Sign up with company details using k@kanielt.com
3. Approval is typically **instant**
4. Activate benefits — credits, OpenAI access, and tools unlock automatically
5. Complete verification tasks for full $25K credit unlock
6. 45 days before expiration, automatic review for extensions

### Upgrade Path to $150K
- After joining Founders Hub, complete engagement milestones
- Apply for Microsoft for Startups program for additional credits
- May need partner/investor referral for full $150K tier
