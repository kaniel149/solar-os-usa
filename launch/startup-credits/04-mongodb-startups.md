# MongoDB for Startups

## Program URL
https://www.mongodb.com/solutions/startups

## What You Get
- MongoDB Atlas credits (amount varies, recently increased 50%)
- Voyage AI tokens for embedding and reranking models
- Matched credits from partners (Fireworks AI, Temporal)
- One-on-one technical advice
- Co-marketing opportunities
- Professional services access

## Eligibility
- Less than 7 years old
- Series A or earlier
- Building a scalable software product (not agencies or dev shops)
- New to the program
- Active company website
- LinkedIn company profile
- Existing MongoDB Atlas users are eligible

## Application Steps
1. Go to https://www.mongodb.com/solutions/startups
2. Click "Apply Now"
3. Fill out application with company details
4. Submit and wait for welcome email (within 5 business days)
5. Receive unique promo code via email
6. Apply promo code to MongoDB Atlas account
7. Opt-in for expanded partnership credits

## What to Write in the Application

**Company Description:**
KANIEL TORD LLC builds Solar OS, an all-in-one platform for solar companies. We manage complex data relationships — customers, projects, proposals, installations, equipment, crews — making a flexible document database ideal for our data model.

**Why MongoDB:**
Solar OS manages diverse, nested data structures (project timelines, equipment specs, proposal configurations, installation checklists). MongoDB's document model is a natural fit for our schema-flexible requirements. We also plan to leverage MongoDB Atlas Search for full-text search across proposals and customer records, and Vector Search for AI-powered document retrieval.

**Technical Stack:**
React + TypeScript frontend, Node.js backend, currently on Supabase (PostgreSQL). Evaluating MongoDB Atlas for specific microservices that benefit from document-oriented storage, particularly proposal management and project tracking modules.

## Priority: MEDIUM
- Good to have for specific features (search, AI embeddings)
- Apply after core cloud credits are secured
