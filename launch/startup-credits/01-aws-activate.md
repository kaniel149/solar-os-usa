# AWS Activate

## Program URL
https://aws.amazon.com/startups/credits

## What You Get
- **Founders Package (no VC required):** $1,000–$2,000 in AWS credits + 1 year AWS Business Support
- **Portfolio Package (VC/accelerator required):** $5,000–$100,000 in credits + priority support + architecture reviews

## Eligibility
- Pre-Series B startup
- Founded within last 10 years
- New to equivalent AWS Activate credits
- Fully functioning company website
- Active AWS account on paid tier
- Corporate business email (must match company domain — use k@kanielt.com)

### Founders Package (our path)
- Self-funded or pre-Series B
- No prior AWS Activate Founders credits
- No VC affiliation needed

### Portfolio Package (future, if we join an accelerator)
- Must be affiliated with an approved AWS Activate Provider (Y Combinator, Techstars, etc.)
- Need their confidential Organization ID (Org ID)

## Application Steps
1. Create AWS Builder ID with k@kanielt.com
2. Create/verify AWS account on paid tier (add credit card)
3. Go to https://aws.amazon.com/startups/credits
4. Complete AWS Activate profile:
   - Company name: KANIEL TORD LLC
   - Product description
   - Market/traction info
   - Funding stage: Pre-seed / Self-funded
5. Select **Founders Package**
6. Link primary AWS account and submit
7. Wait for validation (typically 1-2 weeks)

## What to Write in the Application

**Company Description:**
KANIEL TORD LLC builds Solar OS, an all-in-one platform for the solar energy industry. Solar OS combines CRM, project management, proposal generation, and installation tracking into a single platform purpose-built for solar companies.

**Product Description:**
Solar OS is a SaaS platform that streamlines operations for solar installation companies. Built with React, TypeScript, and cloud-native architecture, it handles the full lifecycle from lead capture through installation completion. Currently MVP stage with plans to scale across the US solar market.

**Why AWS:**
We plan to use AWS for compute (Lambda/EC2), storage (S3), database services, and AI/ML capabilities to power our proposal generation and energy production estimation features. AWS's reliability and global infrastructure align with our growth plans.

**Market Opportunity:**
The US residential solar market is projected to grow significantly. Solar companies currently use fragmented tools — separate CRM, project management, and proposal tools. Solar OS consolidates everything into one platform.
